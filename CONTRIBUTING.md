# Contributing to Meshtastic Node Monitor

Grazie per il tuo interesse nel contribuire a questo progetto! 

## Come Contribuire

### Segnalare Bug

1. Verifica che il bug non sia già stato segnalato
2. Apri una nuova issue con:
   - Descrizione chiara del problema
   - Passi per riprodurre
   - Comportamento atteso vs attuale
   - Browser e versione utilizzati
   - Screenshot (se applicabile)

### Suggerire Funzionalità

1. Apri una issue con label `enhancement`
2. Descrivi la funzionalità e il caso d'uso
3. Spiega perché sarebbe utile

### Pull Requests

1. Fork il repository
2. Crea un branch: `git checkout -b feature/nome-funzione`
3. Fai le tue modifiche
4. Committa: `git commit -am 'Aggiunta funzione X'`
5. Push: `git push origin feature/nome-funzione`
6. Apri una Pull Request

## Linee Guida Codice

### HTML
- Usa semantic HTML5
- Mantieni l'indentazione a 4 spazi
- Commenta sezioni complesse

### CSS
- Usa variabili CSS per i colori
- Prefissa classi specifiche: `.mesh-`, `.monitor-`
- Supporta responsive design (mobile-first)

### JavaScript
- Usa `const` e `let`, evita `var`
- Funzioni con nomi descrittivi
- Commenta funzioni complesse
- Gestisci errori con try/catch

## Testing

Prima di inviare una PR:
1. Testa su diversi browser (Chrome, Firefox, Safari)
2. Verifica il responsive design
3. Controlla la console per errori

## Domande?

Unisciti al [Discord Meshtastic](https://discord.gg/meshtastic) o apri una issue.
